# Dungeons and Druddigons

Dungeons & Druddigons is an unofficial, fan-made homebrew adaptation.

## Development

1. Install dependencies: `npm install`
2. Run local dev server: `npm run dev`
3. Build for production: `npm run build`
4. Preview production build: `npm run preview`

## Deployment

This repo deploys to GitHub Pages automatically from `main` using the workflow in `.github/workflows/deploy-pages.yml`.
